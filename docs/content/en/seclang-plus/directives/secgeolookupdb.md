**Description:** Configure the geolocation database file path for the @geoLookup operator.

**Syntax:** `SecGeoLookupDb PATH`

**Case Sensitive:** Yes

**Implemented:** No

**Example:**

```apache
SecGeoLookupDb /usr/share/GeoIP/GeoLite2-Country.mmdb
```
