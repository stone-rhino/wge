+++

**Description:** Configure the cookie parsing format version.

**Syntax:** `SecCookieFormat VERSION`

**Default:** 0

**Options:**

- **0:** Use Netscape cookie format (version 0)
- **1:** Use RFC 2109 cookie format (version 1)

**Example:**

```
SecCookieFormat 0
```

**Case Sensitive:** Yes
