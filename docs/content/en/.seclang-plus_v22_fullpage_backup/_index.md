First, thanks to the [ModSecurity](https://www.modsecurity.org/) project, which provides a solid foundation for WGE's SecLang++. SecLang is a domain-specific language (DSL) developed by ModSecurity for defining web application protection rules.
SecLang++ extends SecLang with many new features and improvements to meet the needs of modern web application protection.

In this chapter, we will introduce the basic syntax and usage of SecLang, as well as the new features and characteristics added in SecLang++.
