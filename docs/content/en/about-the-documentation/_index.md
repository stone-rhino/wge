The WGE documentation aims to provide users with comprehensive information about installing, integrating, and using WGE. Whether you are a beginner or an experienced user, this documentation will help you make the most of WGE's features.

The WGE documentation covers the following main sections:

- [Introduction](introduction/): This section provides an overview of what WGE is, its architecture, main features, and the benefits of using WGE.
- [Getting Started](getting-started/): This section provides step-by-step guides for installing and integrating WGE.
- [SecLang++](seclang-plus/): This section provides detailed information about various directives and their usage in the SecLang++ language.
- [Version History](version-history/): This section documents WGE's version updates, bug fixes, and new features.

### Contributing to Documentation

This documentation is maintained by the WGE community, and contributions and feedback are welcome. To learn how to contribute to the documentation, please refer to the [Contributing to Documentation](contributing/) section.
