要构建 WGE，您需要一个适合 C++ 开发的环境。我们推荐的开发环境配置如下：
- **操作系统**：Windows 10/11 (64位) + WSL2(Ubuntu 20.04)
- **编译器**：GCC 13.2.0
- **代码编辑器**：Visual Studio Code + 相关插件（C++、CMake、Ragel等）
- **构建工具**：CMake 3.28 
- **包管理工具**：vcpkg
- **其他依赖**：Ragel 6.10、JDK 21、ANTLR4 4.13.2

如果您是一位经验丰富的开发者，可能已经有了自己的偏好配置。但对于初学者或希望快速上手的用户，我们建议使用上述配置，这有助于社区用户跟 WGE 开发团队保持一致，方便交流和协作。况且，我们不想让环境配置成为您开始使用 WGE 的障碍，我们期望您能专注于 WGE 的原理理解和功能实现，而不是在环境配置上花费过多时间。
接下来，我们将按照推荐的配置，一步步帮助您完成开发环境的搭建。
