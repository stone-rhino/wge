WGE 文档旨在为用户提供有关安装、集成和使用 WGE 的全面信息。无论您是初学者还是有经验的用户，这些文档都将帮助您充分利用 WGE 的功能。
WGE 文档涵盖以下主要部分：
- [概述](introduction/)：本节概述 WGE 是什么、架构概述、主要功能以及使用 WGE 的好处。
- [入门](getting-started/)：本节提供安装和集成 WGE 的分步指南。
- [SecLang++](seclang-plus/)：本节详细介绍 SecLang++ 语言的各种指令及其用法。
- [版本历史](version-history/)：本节记录 WGE 的版本更新、修复的错误和新增功能。
