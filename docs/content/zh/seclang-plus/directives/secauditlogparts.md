title = "SecAuditLogParts"
weight = 56
+++

**描述:** 配置应在审计日志中记录事务的哪些部分。


**语法:** `SecAuditLogParts PARTS`


**默认值:** ABCFHZ

**可选部分:**


- **A:** 审计日志头（必需）
- **B:** 请求头
- **C:** 请求体（仅当启用请求体访问且请求体存在时）
- **D:** 保留用于中间响应头
- **E:** 响应体（仅当启用响应体访问且响应体存在时）
- **F:** 最终响应头
- **G:** 保留用于实际响应体
- **H:** 审计日志尾部（包含附加数据）
- **I:** 精简请求体（C 的替代品，排除文件）
- **J:** 保留用于上传文件信息
- **K:** 所有匹配规则的列表
- **Z:** 最终边界标记（必需）

**区分大小写:** 是

**是否实现:** 否


**示例:**


```apache
SecAuditLogParts ABCFHZ
```
